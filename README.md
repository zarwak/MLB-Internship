## Day 1 — Student Grading System

A small command-line program that asks for a student's name and class, then lets you enter marks for as many subjects as you want. Once you're done, it averages the marks and converts that average into a letter grade (A down to F).

Run it with:

```bash
python Day1/grading.py
```
